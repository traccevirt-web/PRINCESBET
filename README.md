# Carpeta de Imágenes — PRINCES BET

Coloca aquí tus imágenes con los siguientes nombres exactos.
El sitio las cargará automáticamente.

## Imagen principal (Hero)
- `hero.jpg`          → Foto principal que aparece en la sección de inicio

## Foto de Acerca de (About)
- `about.jpg`         → Foto del lado izquierdo en la sección "Sobre mí"

## Galería / Book (6 fotos)
- `image1.jpg`
- `image2.jpg`
- `image3.jpg`
- `image4.jpg`
- `image5.jpg`
- `image6.jpg`

## Formato recomendado
- Formato: JPG o WEBP
- Hero y About: orientación vertical (retrato), mínimo 800×1100 px
- Galería: orientación vertical (retrato), mínimo 600×800 px
- Peso máximo por imagen: 500 KB (optimizar antes de subir)
